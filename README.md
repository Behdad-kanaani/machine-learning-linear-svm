## ğŸ“– Python Code Example

```python
from svm import LinSvm
import numpy as np

# Sample dataset
X = np.array([[2,2], [3,3], [-2,-2], [-3,-3]])
y = np.array([1, 1, -1, -1])

# Train classifier
clf = LinSvm(C=1.0, lr=0.05, epochs=100, bsize=32)
clf.fit(X, y)
clf.fitplatt(X, y)

# Predictions
print(clf.predict([[0,0], [2,2]]))
print(clf.proba([[0,0], [2,2]]))
```
